# Wildfire Hex Plot

### Run with pip
Navigate to the project root
> pip install -r requirements.txt
> 
> python app.py

Open http://127.0.0.1:8050/ 


### Run with pipenv
> pipenv install
> 
> pipenv run python app.py

Open http://127.0.0.1:8050/ 


### Contributing

#### Tests
Tests must go into the tests folder
Test files must be of form:
> test_X.py
